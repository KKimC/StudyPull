# StudyPull
SNS for stududent
## 함께 공부하고 싶은 사람들을 위한 앱 입니다.
기본적인 기능으로는 자신이 궁금한 문제나 공부진행상황을 실시간으로 피드에 올려 즉각적인 피드백을 받을수 있게하고, 그 외에도 유저간 대화기능 등을 지원예정중에 있는 앱입니다.
